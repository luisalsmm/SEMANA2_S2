Solucion de ejercicio 2

Definicion y ejemplo de nuevas etiquetas

¿Que es el HEADER?

Respuesta: Muestra el primer contenido relacionado al ancestro mas cercano

¿Que es el FOOTER?

Respuesta: Muestra el pie de pagina del contenido

¿Que es el NAV?

Respuesta: Muestra el menu de navegacion "ul"-->"li"

¿Que es el MAIN?

Respuesta: Engloba el contenido principal

¿Que es el SECTION?

Respuesta: Muestra una seccion de un contenido

¿Que es el ARTICLE?

Respuesta: Muestra a detalle el contenido
